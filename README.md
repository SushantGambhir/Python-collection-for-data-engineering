# Python-collection-for-data-engineering

This repository contains a progressive learning collection of 8 Jupyter notebooks that teach practical Python data engineering skills. The content starts with basic file I/O operations and advances through JSON processing, Pandas fundamentals, data aggregation, joins, sorting, and data export. It uses a realistic retail e-commerce dataset (with 6 related tables: departments, categories, products, customers, orders, and order_items) to demonstrate real-world data processing scenarios.

The notebooks follow a clear learning progression:

01-03: Core Python file/data handling (I/O, lists, JSON)

04-08: Pandas library mastery (reading, transforming, joining, sorting, exporting data)
